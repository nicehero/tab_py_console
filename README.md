# Easy python process and console manager gui by tkinter

Only for windows
## Feature
#### Support Drag file into gui to run
#### Support minimize to system tray
#### Support utf-8 gbk output
#### Support tqdm output

## Installation
```shell
pip3 -r requirements.txt
```
## Easy to use 
### double click tab_py_console.pyw Go!
